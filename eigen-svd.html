2D Matrix Transformations

Singular Value Decomposition (SVD)

For any matrix A ∈ ℝm×n, SVD factorises it as:

A = U Σ Vᵀ

U (m×m orthogonal) = left singular vectors; Σ = diagonal matrix with σ₁ ≥ σ₂ ≥ 0; V (n×n orthogonal) = right singular vectors.

Geometric Interpretation

The unit circle maps to an ellipse. Right singular vectors (V, solid yellow) give the principal axes of the ellipse in the input space. The singular values give the semi-axis lengths. Left singular vectors (U, dashed blue) give the axes in the output space.

Eigenvalues

For square A, eigenvalues λ and eigenvectors v satisfy Av = λv. Found via det(A − λI) = 0. For 2×2: λ² − tr(A)λ + det(A) = 0.

Spectral Theorem

For symmetric A = Aᵀ: all eigenvalues are real, eigenvectors are orthogonal, and A = QΛQᵀ. For symmetric A, SVD and eigendecomposition coincide.

SVD ↔ Eigenvalues

AᵀA = VΣᵀΣVᵀ, so right singular vectors V are eigenvectors of AᵀA with eigenvalues σᵢ². Thus σᵢ(A) = √λᵢ(AᵀA).

Applications

  • PCA — SVD on centred data gives principal components
  • Low-rank approximation (Eckart–Young theorem)
  • Numerically stable solving of linear systems

3D Matrix Transformations

3D Singular Value Decomposition

For A ∈ ℝ³ˣ³, SVD is A = UΣVᵀ where U, V are 3×3 orthogonal matrices and Σ = diag(σ₁, σ₂, σ₃) with σ₁ ≥ σ₂ ≥ σ₃ ≥ 0.

Geometric Picture

The unit sphere maps to an ellipsoid. The three singular values give the semi-axis lengths. Right singular vectors (V, yellow arrows) show the principal input directions; left singular vectors (U) give the output directions.

Volume Transformation

Volume of ellipsoid = (4/3)π σ₁σ₂σ₃ = (4/3)π |det A|

3D Eigenvalues

Characteristic polynomial has degree 3. A symmetric 3×3 matrix always has 3 real eigenvalues (Spectral Theorem). General matrices may have one real + two complex conjugate eigenvalues, indicating rotation.

Special Cases

  • Orthogonal (QᵀQ = I): all σᵢ = 1 — pure rotation/reflection
  • Rank-deficient: one or more σᵢ = 0 — sphere collapses to lower dimension
  • Symmetric: SVD and eigendecomposition coincide — A = QΛQᵀ
Implementation Note

Eigenvalues are computed exactly for symmetric matrices via Jacobi iteration on A. For non-symmetric matrices, Cardano's formula gives the characteristic roots. Singular values always use Jacobi on AᵀA.
