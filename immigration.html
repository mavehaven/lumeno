Supabase Integration
First time? New to Supabase? Create a free project at supabase.com, then go to Project Settings → API to find your URL and anon key. After connecting, click "Run Setup SQL" below to create all tables automatically.
How it works
• Every section save writes to local SQLite and syncs to Supabase via REST API.
• The anon key is stored in sessionStorage only — never sent anywhere except your own Supabase project.
• Row Level Security is disabled by default in the setup SQL so the form can write without auth. Enable RLS and add policies once you add user authentication.
